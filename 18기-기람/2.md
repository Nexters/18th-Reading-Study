##### 4) 가변값

기본형 데이터 - 불변값

참조형 데이터 - 가변값

기본형 데이터 - 참조형 데이터 차이:

객체의 변수(프로퍼티) 영역이 별도로 존재하는 것

```javascript
var a = 10;
var b = a;

var obj1 = {c: 10, d: 'd'};
var obj2 = obj1;

b = 15;
obj.c = 20;
```

이 때, 변수 a와 b는 서로 다른 메모리 주소를 가지게 됨

그러나 변수 obj1, obj2는 여전히 같은 메모리 주소를 가짐

하지만 obj1, obj2의 경우

```javascript
var obj1 = {c: 10, d: 'd'};
var obj2 = obj1;

obj2 = {c: 20, d:'d'};
```

라고 했을 때, 아예 새로운 객체를 할당했으므로 다른 주소를 갖게 됨.
